# habitee - Mobile

FlutterとFlameで実装された習慣継続サポートアプリ

## セットアップ

```bash
flutter pub get
```

## 実行

```bash
flutter run
```

## 機能

- カテゴリー選択（ギャンブル、お酒、タバコ、ゲーム、買い物、SNS）
- 習慣の継続日数を可視化
- 成長段階に応じたキャラクターアニメーション
- ローカルストレージでのデータ保存

## 成長システム

日数に応じてキャラクターが成長します：
- 🌱 種 (0-6日)
- 🌿 芽 (7-29日)
- 🍀 若葉 (30-89日)
- 🌳 成長 (90-179日)
- 🌸 開花 (180-364日)
- 🌺 満開 (365日以上)

## 使用技術

- Flutter: UIフレームワーク
- Flame: ゲームエンジン（キャラクターアニメーション）
- SharedPreferences: ローカルデータ保存
